# ainvest.com.tr — statik site

Türkçe bilgilendirme: yapay zeka ve yatırım araştırması, BIST bağlamı. **Yatırım tavsiyesi değildir.**

## Yerel önizleme

```bash
python3 -m http.server 8080
```

## Yayın ve özel alan

Statik dosyalar, depo kökündeki otomatik dağıtım iş akışı ile yayımlanır.

Özel alan için kök (`@`) DNS’e genelde dört **A** kaydı eklenir; güncel adresleri barındırma sağlayıcınızın güncel dokümantasyonundan doğrulayın. `CNAME` dosyası kök alan adınızla eşleşmelidir.
